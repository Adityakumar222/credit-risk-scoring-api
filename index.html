What this API does

End-to-end ML pipeline from raw data to real-time risk scoring via a clean REST endpoint.

ML Risk Scoring

Logistic Regression model trained on cleaned, feature-engineered loan data to predict default probability.

⚡

FastAPI Endpoint

Submit applicant data as JSON, get back risk level and probability in milliseconds. Auto-generated Swagger docs included.

Income ratios, credit history encoding, and loan term normalization — all handled in the pipeline before prediction.

🐳

Dockerized & Deployed

Containerized with Docker, deployed on Render. Ready for integration into any lending workflow or dashboard.
